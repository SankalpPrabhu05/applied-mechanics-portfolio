# FEM: Solids — TME250

Chalmers University of Technology, Academic year 2025/2026, SP2.
Author: Sankalp Manojkumar Prabhu.

Three computer assignments, each in its own subfolder since they cover distinct topics:

- [`CA1-incompressible-elasticity/`](./CA1-incompressible-elasticity) — analytical limiting-case bounds, CST elements under nearly-incompressible loading (ν → 0.5), and a Taylor-Hood mixed formulation for stable incompressible response.
- [`CA2-contact-mechanics/`](./CA2-contact-mechanics) — Lagrange-multiplier contact with active-set iteration, quasi-static indentation, and implicit vs. explicit dynamic contact.
- [`CA3-adaptive-mesh-refinement/`](./CA3-adaptive-mesh-refinement) — manufactured-solution reference for exact energy norm, convergence study of linear vs. quadratic elements, an energy-norm error estimator with effectivity index, and threshold-based adaptive mesh refinement vs. uniform refinement.


See each subfolder's own README for dependency notes, what's included vs. excluded, and how to read the code.
